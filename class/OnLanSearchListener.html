  • 这个类负责接受内网搜索回调

  • 参数 ：
    • gid ： 设备gid
    • ip ： 设备内网ip
  • 返回值： 无

  • 作用 ： 当搜索完后调用此函数，子类需实现此函数
  • 参数 ： 无
  • 返回值： 无
virtual void onSearchFinish() = 0;
